## עברית

### מבוא
תוסף "חוסם פרסומות בחיפוש גוגל" (להלן: "התוסף") מחויב להגנה על פרטיות המשתמשים. מדיניות פרטיות זו מסבירה כיצד התוסף פועל וכיצד הוא מטפל במידע.

### איסוף מידע
התוסף **אינו אוסף, אינו שומר ואינו משדר** כל מידע אישי או נתוני גלישה. התוסף פועל באופן מקומי בלבד על המכשיר שלך.

### אופן הפעולה
התוסף פועל אך ורק בדפי תוצאות חיפוש של Google. הוא מזהה ומסיר אלמנטים של תוצאות ממומנות ופרסומות מתוך דף ה-HTML באופן מקומי בדפדפן שלך. לא מתבצעת שום תקשורת עם שרתים חיצוניים.

### הרשאות
התוסף מבקש גישה לדפי חיפוש של Google בלבד (כגון www.google.com, www.google.co.il ודומיינים מקומיים נוספים). הרשאה זו נדרשת אך ורק לצורך הסרת תוצאות ממומנות מדף החיפוש.

### קוד מרוחק
התוסף אינו טוען ואינו מריץ קוד מרוחק. כל הקוד ארוז בתוך התוסף ופועל באופן מקומי בלבד.

### שיתוף מידע עם צדדים שלישיים
התוסף אינו משתף מידע עם צדדים שלישיים, מכיוון שאינו אוסף מידע כלל.

### שינויים במדיניות
אם תבוצע תמורה במדיניות פרטיות זו, השינוי יפורסם בדף זה עם תאריך עדכון חדש.

### יצירת קשר
לשאלות בנוגע למדיניות פרטיות זו, ניתן לפתוח Issue במאגר ה-GitHub של הפרויקט.

## English

### Introduction
The "Google Search Ad Blocker" extension (the "Extension") is committed to protecting user privacy. This privacy policy explains how the Extension operates and how it handles information.

### Data Collection
The Extension **does not collect, store, or transmit** any personal information or browsing data. It operates entirely locally on your device.

### How It Works
The Extension operates exclusively on Google Search result pages. It identifies and removes sponsored result and advertisement elements from the HTML page locally in your browser. No communication with external servers takes place.

### Permissions
The Extension requests access to Google Search pages only (such as www.google.com, www.google.co.il, and other local domains). This permission is required solely for the purpose of removing sponsored results from the search page.

### Remote Code
The Extension does not load or execute any remote code. All code is bundled within the extension and runs locally only.

### Third-Party Sharing
The Extension does not share data with any third parties, as it does not collect any data.

### Changes to This Policy
If this privacy policy is modified, the change will be published on this page with an updated date.

### Contact
For questions regarding this privacy policy, please open an Issue in the project's GitHub repository.
