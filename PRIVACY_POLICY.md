## עברית

### מבוא
תוסף "חוסם פרסומות בחיפוש גוגל" (להלן: "התוסף") מחויב להגנה על פרטיות המשתמשים. מדיניות פרטיות זו מסבירה כיצד התוסף פועל וכיצד הוא מטפל במידע.

### איסוף מידע
התוסף **אינו אוסף, אינו שומר ואינו משדר** כל מידע אישי או נתוני גלישה. התוסף פועל באופן מקומי בלבד על המכשיר שלך.

### אופן הפעולה
התוסף פועל בדפי תוצאות חיפוש של Google ובתיבת הדואר של Gmail. הוא מזהה ומסיר אלמנטים של תוצאות ממומנות, אימיילים ממומנים ובאנרים פרסומיים מתוך דף ה-HTML באופן מקומי בדפדפן שלך.

### עדכון רשימות החסימה
כדי לשמור על יעילות החסימה, התוסף מוריד מעת לעת קובץ רשימות חסימה (`filters.json`) משרת GitHub הציבורי (raw.githubusercontent.com). מדובר בהורדה של **רשימת סלקטורים בלבד (נתונים)** — לא נשלח שום מידע אישי או נתוני גלישה בבקשה זו, והתוסף אינו מוריד ואינו מריץ קוד.

### הרשאות
התוסף מבקש גישה לדפי חיפוש של Google ול-Gmail בלבד, וכן הרשאות `storage` (לשמירת רשימות החסימה מקומית) ו-`alarms` (לתזמון בדיקת עדכון תקופתית). אין גישה לתוכן ההודעות שלך — התוסף רק מסיר אלמנטים פרסומיים מהדף.

### קוד מרוחק
התוסף אינו טוען ואינו מריץ קוד מרוחק. כל הקוד ארוז בתוך התוסף. מהשרת מורדים אך ורק נתונים (רשימת סלקטורים בפורמט JSON) ולא קוד הניתן להרצה.

### שיתוף מידע עם צדדים שלישיים
התוסף אינו משתף מידע עם צדדים שלישיים, מכיוון שאינו אוסף מידע כלל.

### שינויים במדיניות
אם תבוצע תמורה במדיניות פרטיות זו, השינוי יפורסם בדף זה עם תאריך עדכון חדש.

### יצירת קשר
לשאלות בנוגע למדיניות פרטיות זו, ניתן לפתוח Issue במאגר ה-GitHub של הפרויקט.

## English

### Introduction
The "Google Search Ad Blocker" extension (the "Extension") is committed to protecting user privacy. This privacy policy explains how the Extension operates and how it handles information.

### Data Collection
The Extension **does not collect, store, or transmit** any personal information or browsing data. It operates entirely locally on your device.

### How It Works
The Extension operates on Google Search result pages and in Gmail. It identifies and removes sponsored results, sponsored emails, and promotional banner elements from the HTML page locally in your browser.

### Block List Updates
To keep blocking effective over time, the Extension periodically downloads a block-list file (`filters.json`) from the public GitHub server (raw.githubusercontent.com). This is a download of **a list of selectors (data) only** — no personal information or browsing data is sent with this request, and no code is downloaded or executed.

### Permissions
The Extension requests access to Google Search and Gmail pages only, plus the `storage` permission (to cache the block list locally) and the `alarms` permission (to schedule periodic update checks). It has no access to the content of your messages — it only removes advertising elements from the page.

### Remote Code
The Extension does not load or execute any remote code. All code is bundled within the extension. Only data (a JSON list of selectors) is downloaded from the server — never executable code.

### Third-Party Sharing
The Extension does not share data with any third parties, as it does not collect any data.

### Changes to This Policy
If this privacy policy is modified, the change will be published on this page with an updated date.

### Contact
For questions regarding this privacy policy, please open an Issue in the project's GitHub repository.
